# Controller Code for the ROV
---
## Dependencies:
- inputs
- socket
## Setting up a virtual environment:
```
python3 -m venv .venv
```
## *nix:
```
source .venv/bin/activate
```
## Windows (idk):
```
./.venv/bin/activate.sh
```
## Installing Libraries
```
python3 -m pip install -r requirements.txt
```
## Connecting to Raspberry Pi
```
python3 connect.py
```
